### ğŸ”¹ 1. Clone the Repository

```bash

```

### ğŸ”¹ 2. Setup Backend

```bash

```

- Create a .env file inside /backend based on [.env.example](./backend/.env.example)

```bash

```

- The backend will run at: [http://localhost:5000](http://localhost:5000)

### ğŸ”¹ 3. Setup Frontend
